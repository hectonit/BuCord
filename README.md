# BuCord

This is my first discord bot with simple economics, games and some interesting commands.

## Commands

This bot has four sections of commands:

* **economy**
    * balance
    * top
    * bet
* **games**
    * tictactoe
* **general**
    * user_card
    * course
    * bot_info
    * ping
    * help
    * find_bug
* **moderation**
    * change_prefix
    * goodbye
    * welcome
    * give
